# LeetCode-POTD
A curated collection of my daily solutions to LeetCode's Problem of the Day. Focused on optimized algorithms, clean C++ implementation, and consistent daily coding practice.

## 🛠️ Tech Stack
- **Language:** C++
- **Platform:** [LeetCode](https://leetcode.com/problemset/all/)
- **Paradigm:** Optimized Algorithms (Greedy, DP, Graphs, etc.)

## 📝 Approach
Each solution is implemented with a focus on:
1. **Time Complexity:** Aiming for the most optimal theoretical bound.
2. **Space Complexity:** Minimizing memory footprint.
3. **Readability:** Clean, standard C++ practices.
